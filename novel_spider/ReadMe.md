## Novel Spider

spider for chinese online novel's information fetch.

## How TO
* install scrapy
```shell
pip install scrapy
```
* install chromedriver  
> download and install  [chromedriver](https://sites.google.com/a/chromium.org/chromedriver/downloads) .

* run script
```shell
cd qidian
scrapy crawl qidian -o qidian.json
```
